## Usage

You may use these icons to label content with Made Mark authorship disclosure.

**Permitted:**
- Display icons alongside content labels
- Include in websites, documents, and publications
- Use in tools implementing Made Mark

**Not Permitted:**
- Modify, adapt, or create derivatives of the icons
- Use icons in ways that misrepresent authorship
- Incorporate into product names or branding

See [Governance](https://mademark.org/governance) for full terms.
