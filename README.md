specification.

## The three marks

| Mark | Meaning |
|---|---|
| **Human Made** | Conceived and produced by people. No AI tools involved in the creative work itself. |
| **Human Designed, AI Made** | Human creative direction, AI production. A person shaped the intent — AI produced the artifact. |
| **AI Made** | Conceived and produced primarily by AI, with no human creative direction for the specific output. |

Full definitions, badge embed snippets, and usage guidance:
[mademark.org/marks](https://mademark.org/marks/)

## Permitted uses

- Displaying an icon next to content it labels
- Including an icon in a Made Mark badge, footer credit, or physical marking
- Referencing an icon in documentation about implementing Made Mark

## Restricted

- Modifying, recoloring, or restyling the icons
- Using an icon to imply authorship disclosure that hasn't actually been made
- Using an icon as part of unrelated product branding
